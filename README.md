# BloomBuddyWeb
A cross-platform habit tracking application that gamifies daily routines through plant growth, progress tracking, and streak management. Originally built for Android and now being reimagined as a modern web application.

Tech Stack:
React 19
TypeScript
Vite
React Router
Tailwind CSS
Firebase (or Supabase)
TanStack Query
Zustand
Recharts
Framer Motion

Contributors: Mollie P. & Travis M.
