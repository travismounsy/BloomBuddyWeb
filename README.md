Bloom Buddy

Team Members: Travis Mounsy & Mollie Pathammavong

Bloom Buddy is a plant-themed habit tracking app designed to help users build and maintain positive daily routines. Users can create, edit, and delete habits such as studying, exercising, or staying hydrated, and check them off each day. Each habit is represented by a virtual plant that grows through stages as tasks are completed. The plant maintains its growth level based on overall progress and updates if habits are added or removed. The app will also display daily progress and streak information to help users stay motivated. All data will be stored locally on the device for quick access and offline use. Bloom Buddy combines simple habit management with visual growth to encourage consistency and long-term personal development.

Git Workflow

Git branching and commit convention:

Every feature branch updates code and docs.  
Commit messages follow Conventional Commits, for example:  
feat(habits): add recurring habit scheduling  
docs(system): document recurring habit workflow  
refactor(layout): simplify navigation shell  
test(progress): add streak calculation tests  

This makes the project history easy to follow.
